# C语言基本数据类型大小

|     类型      | 16位机器 | 32位机器 | 64位机器 |
| :-----------: | :------: | :------: | :------: |
|   **char**    |    8     |    8     |    8     |
|   **short**   |    16    |    16    |    16    |
|    **int**    |    16    |    32    |    32    |
|   **long**    |    32    |    32    |    64    |
| **long long** |    64    |    64    |    64    |
|   **float**   |    16    |    32    |    32    |
|  **double**   |    64    |    64    |    64    |
